# 🎧 Spotify Settings Backup

Backup and restore your Spotify settings in seconds.

This tool lets you save your Spotify configuration and restore it later — perfect for reinstalling Spotify, switching setups, or experimenting with tools like Spicetify without losing your setup.

---

## 🚀 Why this tool?

Spotify doesn't provide an easy way to backup your settings.

If you've ever:

* reinstalled Spotify and lost your configuration
* spent time setting everything up again
* used Spicetify and had it break after an update

…then this tool solves that.

---

## ✨ Features

* 💾 Backup your Spotify settings
* 🔄 Restore them anytime
* ⚡ Fast and simple
* 🖥️ Runs locally (no server needed)
* 🔒 No data leaves your device

---

## 📦 How to use

1. Download or clone this repository
2. Run the tool
3. Click **Backup** to export your settings
4. Save the file somewhere safe
5. Use **Restore** whenever needed

---

## 📦 Installation

1. 📥 Download or clone this repository  
2. 📂 Move `settingsBackup.full.js` to:

   ```
   %APPDATA%\spicetify\Extensions
   ```
3. ⚙️ Enable the extension:
   ```
   spicetify config extensions settingsBackup.full.js
   spicetify apply
   ```
4. 🔄 Restart Spotify


---


## 💡 Example use cases

* Reinstalling Spotify without losing settings
* Switching between multiple setups
* Testing new configs safely
* Using Spicetify without reconfiguring everything

---

## ⚠️ Important notes

* This tool works locally only
* Your data is never uploaded anywhere
* Keep your backup files safe
